# Sea-Battle

**Sea-Battle** — консольная игра «Морской бой» на C++ для Windows 10/11 (x64).

Проект уже содержит готовый исполняемый файл `Sea-Battle.exe`, который можно запустить сразу, а также скрипт `build.bat` для пересборки игры из исходников с автоматической установкой компилятора при первом запуске.

---

## Быстрый запуск без сборки

1. Скачайте проект и распакуйте его в удобную папку (например, `"C:\Морской бой"`).
2. Откройте проводник и перейдите в папку проекта.
3. Дважды кликните по файлу `Sea-Battle.exe`
